Feedback

The recommended way to get in touch, if you have questions or comments about the Mozilla Archive Format extension, is to write a message to the MAF mailing list.

If you think you have found a bug in MAF, you may also check if the problem has already been reported, then file a new bug in Bugzilla.

You have the following options:

  1. View the MAF mailing list archives.
  2. Subscribe to the MAF mailing list, then post a message to maf@mozdev.org.
  3. Post a message to the public.mozdev.maf newsgroup.
    To prevent spam, access to the newsgroup is password protected. Specify username mozdev and password maduser.
  4. View all open bugs in Bugzilla.
  5. Enter a new bug in Bugzilla.

For more links, see the site's mailing list page and the bugs page.
