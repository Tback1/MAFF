The MAF extension enables Firefox to save complete web pages in a single file, using the following formats:

MAFF (Mozilla Archive Format File)

MAFF files are standard ZIP files containing one or more web pages, images, or other content stored in a platform-independent way. MAFF files also contain metadata like the orirginal URL the page was saved from.

Firefox with the MAF extension can open and save these files seamlessly.

Since MAFF is based on standard ZIP, the contents of MAFF archives can still be accessed on any computer using a ZIP extraction utility or the jar: protocol.

MAFF files use the *.maff or *.maff.zip file extensions.

MHTML (MIME HTML)

MHTML is the file format used by Internet Explorer when saving complete web pages. The MAF extension allows Firefox to open and save these files seamlessly.

A single MHTML file may contain only one web page.

MHTML files use the *.mht or *.mhtml file extensions.

Downloading and installing

You can download and install the latest version of MAF in Firefox very easily.

Current features
  • Save and load complete web pages using either MAFF or MHTML
  • Command to open the original URLs instead of the archived copy
  • Store more than one saved page in a MAFF archive
  • Open the contents of MAFF archives in multiple tabs at once
  • Replace absoulte URLs on web pages if there is a local version in any other open archive
  • Localized in French, Italian, Polish, Portuguese and Russian
