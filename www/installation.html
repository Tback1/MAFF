Installation
Latest version

The latest version of MAF is compatible only with Firefox 3. MAF is still available for older versions of Firefox, but only the latest version will receive improvements, bugfixes and support from the project maintainers.

Download and install: maf-0.7.1_unofficial-fx.xpi - MAF 0.7.1 unofficial for Firefox 3

This release has been tested with Firefox 3.0.1 on Windows XP, but is expected to work on all the platforms supported by Firefox 3.
