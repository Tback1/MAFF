Installation

Install MAF

The latest version of MAF is compatible only with Firefox 3. MAF is still available for older versions of Firefox, but only the latest version will receive improvements, bugfixes and support from the project maintainers.

This release has been tested with Firefox 3.0.1 on Windows XP, but is expected to work on all the platforms supported by Firefox 3.

What's new in the latest version?

MAF has the following user-visible improvements over MAF 0.6.3:

  • Compatibility with Firefox 3 on all operating systems.
  • Clean installation and uninstallation.
  • Redesigned preferences user interface.
  • Temporary files are now created in the system's temporary folder by default.
  • Creating file associations on Windows works again.
  • Stabilized file extension and type handling:
    • Handle only the .maff extension, and not its variations, since double extensions cannot be handled correctly by Firefox.
    • Removed support for the old .maf and .maff.zip extensions.
    • Added support for .mhtml file extension for MHTML archives.
    • MAF does not register applications for its MIME types anymore.
  • Removed features:
    • maf: protocol (use jar: instead).
    • document.write override.
  • Bugfixes:
    • Opening archives from the command-line or by double-clicking a file when no browser windows are visible now works reliably.
    • Now the save functions always overwrite the contents of existing archives, rather than appending.
    • Copying archive metadata to the clipboard works again.

Previous versions

You can download all the previous versions of MAF from the project page on Mozdev.org.

The following are the latest versions available for Firefox 2 and below. If you are using an unofficial version of MAF 0.6.3 on Firefox 2, you can now install version 0.6.3.1. The functionality is identical, except that MAF will be updated to the latest version as soon as Firefox 3 is installed.

Important: these versions are provided as a convenience for Firefox 2 users and not actively developed anymore.

maf-0.6.3.1.xpi - MAF 0.6.3.1 for Linux and Windows

menumaf-0.6.2.1.xpi - MenuMAF 0.6.2.1 for Linux and Windows. This package adds menu entries and shortcut keys for accessibility to the file menu.

The complete changelog for versions prior to these ones can be found here.
