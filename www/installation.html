Installation

Latest XPI - 0.2.17

Installing the XPI is only half of the process. The next step is to setup the scripts that would be used to perform the actual archiving. This only has to be done once. After this is done, the preferences for the MAF extension should be set. Below are ultities and setup instructions for configuring MAF to work with Infozip's Zip and Unzip programs.

Linux utilities setup:
  1. Create a temporary directory /tmp/maftmp.
  2. Place the mafzip.sh and mafunzip.sh scripts in the /tmp directory.
  3. Make the scripts executable (chmod +x maf*.sh in a shell).
  4. Ensure /usr/bin/zip and /usr/bin/unzip are available for the scripts to execute. If the location of either zip or unzip is different, change the scripts.

After the utilities have been setup, complete the configuration by checking the preferences.
