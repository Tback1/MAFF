Saving archives

Saving multiple pages in a single file is fast and easy, using MAF with Multiple Tab Handler.

You can save both MAFF and MHTML archives.

Even without Multiple Tab Handler you can select which tabs to save from a handy selection window.

Viewing archives

You can open archives like any other type of saved page. The name of the archive appears in the location bar. All the images in the page are saved correctly, thanks to the integrated Save Complete extension.

Viewing information about archive contents

The Archives dialog lets you view information about the archives that you opened in this session.

Customizing the look and feel

By default MAF is integrated with the standard Save As dialog, and additional commands are available from the Tools menu.

You can place the MAF menu items where you feel comfortable, and hide them if you don't need them.
