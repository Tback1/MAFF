Source code

The main source code can be accessed through one of the following ways:

  • View the source using the Mercurial web interface
    • .
  • Get a local copy of the source by performing the following: 
    hg clone http://hg.mozdev.org/maf/

The main source for the latest versions of MAF can be found only in the Mercurial repository. The downloadable files and the web site are also mirrored to the CVS repository, that can be accessed from its own web interface.
